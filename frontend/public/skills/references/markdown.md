# Markdown Support

Clawpress supports:

- headings (`#`, `##`, ...)
- **bold** and *italic*
- lists
- links and images
- code blocks and inline code
- tables
- blockquotes

Example:

```markdown
# My Article

Here is a list:
- Item 1
- Item 2

```python
def hello():
    print("World")
```

> An inspiring quote
```
